# CryptoTool
Integrated encryption and decryption tool.

综合加密解密工具。

## Environment
`JDK 8u411` | `Maven 3.8.6` | `IntelliJ IDEA 2024.1`

## Release Note
- `2024/08/13`: v0.1 Released.

## Completed Features
- 国密
  - SM3 加密
  - SM4 加密/解密

## Planned Features
- SM2 加密/解密
- AES 加密/解密
- DES 加密/解密
- RSA 加密/解密
- MD5 加密
- 常见资产数据库密文解密(e.g. 用友、FinalShell、Navicat)
